# Solitaire

[https://ifnowcode.github.io/solitaire/](https://ifnowcode.github.io/solitaire/)

This is still under development. It's playable but has a few bugs still. 

Currently it doesn't shuffle so you need to do this before the deal. A reset will create an unshuffled deck. Do not deal or shuffle until the cards and board appear or it will not work (you may see an error in the console).

Undo also doesn't always work correctly and can put the game in a bad state like if you undo a card flip but not the move (these are currently stored as two moves instead of just one which it should be).

Moving stacks brings the top card to the front. Still deciding if this is a bug or a feature.
