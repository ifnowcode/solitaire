# Solitaire

[https://ifnowcode.gitub.io/solitaire](https://ifnowcode.gitub.io/solitaire)

This is still under development. It's playable but will fail if not used correctly. I got stuck trying to get card stacks to move so you have to do this manually. 

Currently it doesn't shuffle so you need to do this before the deal. A reset will create an unshuffled deck.
